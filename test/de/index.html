Erweiterte Einstellungen
  • Dies ist eine kostenlose Webanwendung zur Stapelkonvertierung von AVIF-Bildern (oder WebP-Bildern usw.) in gängige Bildformate wie JPEG und PNG.
  • It uses a browser's built-in function for the conversions, so no data is sent to any server and therefore it is fast and safe.
  • Die konvertierten Bilder werden als eine Zip-Datei ausgegeben, wenn mehrere Bilder geladen sind.
  • Benötigt die neueste Version von Firefox oder Chrome, um AVIF-Bilder zu laden.
